# Quickstart: Settings Panel and Integrations

1. Start the app with `npm run dev`.
2. Sign in with a GitHub personal access token.
3. Open Settings from the sidebar or user menu.
4. Enter an agent base URL such as `http://localhost:11434/v1`.
5. Leave the API key blank to test fallback behavior, or provide one.
6. Save settings and reload the page.
7. Send a chat prompt and confirm the configured agent is used.
