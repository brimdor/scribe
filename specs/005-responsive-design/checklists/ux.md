# Specification Quality Checklist: UX

**Purpose**: Validate specification completeness before planning

## User Experience Quality
- [ ] CHK001 - Are breakpoints explicitly defined? [Completeness, Spec]
- [ ] CHK002 - Is the behavior of navigation elements (e.g. sidebars vs drawer) defined for each breakpoint? [Clarity, Spec]
- [ ] CHK003 - Are minimum touch target sizes defined? [Measurability, Spec]
- [ ] CHK004 - Is horizontal scrolling explicitly forbidden on small screens? [Requirement Quality, Spec]
- [ ] CHK005 - Are font scaling or typography adjustments mentioned? [Completeness, Spec]
